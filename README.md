# Terraform + Heroku + ExpressJS


## Software
- Terraform
- Heroku
- ExpressJS

## Config

- change name file `terraform.tfvars_EXAMPLE` for `terraform.tfvars`
- change variables of `terraform.tfvars`


## Used

- `terraform init`
- `terraform plan`
- `terraform apply`
- `terraform destroy`
